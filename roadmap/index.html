III — Axioms

Five invariants

Axiom I

Identity is architecture, not prompt

Values with weights, voice parameters, core tensions — evaluated quantitatively on every output.

ICS + VCS computed per generation. Character-breaking outputs blocked.
Axiom II

Memory must consolidate, not accumulate

Context windows are scrollback buffers. Real memory: episodic → summaries → condensed. A DAG, not a list.

DAG consolidation. Hybrid retrieval: vector + keyword + recency + importance.
Axiom III

Evolution must be directed, not entropic

Uncontrolled drift is decay. Creator-directed arcs, trait targets, mood tendencies.

Arcs, epochs, directives, drift rules. Time-series with velocity + trajectory queries.
Axiom IV

Every claim must be grounded

Hallucinating your own history is confabulation. Claims cross-referenced. Contradictions: zero tolerance.

Claim extraction. Citation linking to source memory IDs. Auto-regeneration.
Axiom V

Entities must be economically self-aware

An agent that doesn't understand its economics is subsidized. Revenue, costs, audience — operational constraints.

17,000+ agents on Virtuals. Zero know what they cost to run. LiveClaw agents do.
